# 가상 환경 생성
# 아래 예시는 Conda에서 가상 환경을 생성하는 방법이며, python 3.8의 환경 필요 (3.7 이상)

```bash
$ conda create -n vibe_test python=3.8
$ conda activate vibe_test
$ pip install -r requirements.txt
```
